  • Master degree thesis
    Implementing and modelling a data warehouse to support genomic analyses

    We describe all the steps necessary for the development of a data warehouse system capable of handling standard file formats used in genomics. For genetists is very important the study of genetic variants. It helps the recognition of a possible new genetic disease or the mutation of an existing one. So, creating a data warehouse system is the best way to handle such information and to have the necessary flexiblity to develop all the needs for a complete analysis. Then we discuss the different levels of development of a data warehouse from the process of extracting information from data sources to the ETL process, with the description of the various steps of an OLAP cube. We described a methodology for extracting useful information from a DAG and modify this data structure into a tree in order to be used in a OLAP cube. Finally we report the methods necessary to develop a tool supporting a guided explo- ration of the data warehouse with the explaination of its contents.
